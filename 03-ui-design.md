## 13. 返回按钮组件 (Back Button) - 新增

**添加时间**: 2026-03-17  
**问题来源**: UI-002 - 二级菜单无法返回主页

### 使用场景
所有二级页面（行星、科技、探险、成就、设置、关于）顶部必须包含返回按钮。

### 设计规范
```css
.back-btn {
  padding: 8px 16px;
  background: rgba(255, 255, 255, 0.1);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 6px;
  color: #B0B0B0;
  font-size: 14px;
  cursor: pointer;
}

.back-btn:hover {
  background: rgba(255, 255, 255, 0.2);
  color: #FFFFFF;
}
```

### 布局位置
```
┌─────────────────────────────────────────┐
│ [← 返回]  页面标题                       │  <- panel-header
├─────────────────────────────────────────┤
│                                         │
│  内容区域                                │
│                                         │
└─────────────────────────────────────────┘
```

### 实现文件
- `src/ui/BackButton.tsx` - 组件
- `src/App.css` - 样式

## 14. 存档选择界面 (Save Slot Panel) - 新增

**添加时间**: 2026-03-17  
**问题来源**: SAVE-001, SAVE-002 - 10档位存档系统

### 布局
```
┌─────────────────────────────────────────┐
│ [← 返回]    📂 读取存档                  │
├─────────────────────────────────────────┤
│ ┌────────┐ ┌────────┐ ┌────────┐       │
│ │档位 1  │ │档位 2  │ │档位 3  │ ...    │  (2行5列网格)
│ │我的存档│ │ [空]   │ │ [空]   │        │
│ │💰 5k  │ │        │ │        │        │
│ │🔬 100 │ │        │ │        │        │
│ │🏆 5   │ │        │ │        │        │
│ │⏱ 1h  │ │        │ │        │        │
│ └────────┘ └────────┘ └────────┘       │
├─────────────────────────────────────────┤
│ [读取此存档] [重命名] [删除]             │
└─────────────────────────────────────────┘
```

### 档位卡片设计
```
宽度: 自适应 (网格布局)
最小宽度: 200px
背景: 
  - 空档位: rgba(0, 0, 0, 0.2)
  - 有存档: rgba(0, 206, 209, 0.1)
  - 当前档: rgba(255, 215, 0, 0.1)
边框: 2px solid
  - 默认: rgba(255, 255, 255, 0.1)
  - 选中: #00CED1
圆角: 12px
内边距: 16px
```

### 档位信息展示
```
档位编号: 字体加粗, #00CED1
自定义名称: 16px, 白色, 粗体
货币: 💰 图标 + 数值 (千分位)
研究点: 🔬 图标 + 数值
成就数: 🏆 图标 + 数量
游戏时长: ⏱️ 图标 + "X小时Y分钟"
保存日期: 📅 图标 + 本地化日期
当前标记: 🟡 黄色 "当前" 标签
```

### 操作按钮
```
主按钮 (读取/保存):
  背景: #00CED1
  颜色: #000000
  大小: padding 12px 24px
  
次要按钮 (重命名):
  背景: rgba(255, 255, 255, 0.1)
  边框: 1px solid rgba(255, 255, 255, 0.2)
  
危险按钮 (删除):
  背景: rgba(255, 107, 107, 0.2)
  边框: 1px solid #FF6B6B
  颜色: #FF6B6B
```

### 实现文件
- `src/ui/SaveSlotPanel.tsx` - 组件
- `src/App.css` - 样式
