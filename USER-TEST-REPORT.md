## 发现的问题

### 🔴 严重问题：CSS选择器不兼容

**问题描述**: 新手引导第5步（行星导航）使用了 `:contains()` 选择器
```typescript
// 错误代码
highlightElement: '.nav-btn:contains("行星")'
```

**问题原因**: `:contains()` 是 **jQuery特有的选择器**，不是标准CSS选择器。
浏览器原生 `document.querySelector()` 不支持此语法，导致：
1. 选择器返回 `null`
2. 高亮遮罩无法定位
3. 引导UI可能显示在错误位置
4. 严重情况下JavaScript报错

**影响步骤**: 
- Step 5: 行星导航
- Step 7: 科技导航  
- Step 9: 探险导航
- Step 11: 成就系统

**修复方案**: 使用标准CSS `data-*` 属性选择器
```typescript
// 修复后
highlightElement: '.nav-btn[data-nav="planets"]'
```

**代码修改**:
1. `src/core/TutorialSystem.ts` - 更新4个步骤的选择器
2. `src/App.tsx` - 为导航按钮添加 `data-nav` 属性
