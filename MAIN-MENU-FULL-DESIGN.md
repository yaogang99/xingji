## 按钮功能（全部实现）

### 1. 🎮 开始游戏 / ▶️ 继续游戏
**功能**: 进入游戏主界面
- 无存档时: 显示 "🎮 开始游戏"
- 有存档时: 显示 "▶️ 继续游戏"
- 点击后: 加载游戏数据，进入主界面

### 2. 🔄 新游戏
**功能**: 重新开始（有存档时显示）
- 点击后: 清空当前进度，重新开始
- 覆盖原有存档

### 3. ⚙️ 设置
**功能**: 打开设置面板
- 音量控制（主音量/背景音乐/音效）
- 从主菜单可直接进入设置
- 返回时回到主菜单

### 4. ℹ️ 关于
**功能**: 打开关于页面
- 游戏简介
- 数据统计（30资源/55飞船/33设施/40科技/20成就/8星球）
- 制作信息
- 返回时回到主菜单

## 存档系统（完整实现）

### 自动保存
- 每30秒自动保存
- 保存内容:
  - 货币数量
  - 研究点数量
  - 所有资源数量
  - 已解锁星球
  - 已研究科技
  - 已解锁飞船
  - 已获得成就
  - 音量设置

### 存档检测
- 启动时检测是否有存档
- 主菜单显示存档状态
- 有存档时显示"继续游戏"
- 无存档时显示"开始游戏"

### 存档操作
| 操作 | 功能 |
|------|------|
| 保存 | 自动保存 / 手动保存 |
| 加载 | 读取存档，恢复游戏状态 |
| 删除 | 清除存档（通过新游戏） |
| 导出 | 可导出存档字符串备份 |
| 导入 | 可导入存档字符串恢复 |

## 状态流转

```
启动游戏
    ↓
主菜单（检测存档）
    ├─► 开始游戏 ──► 新游戏
    ├─► 继续游戏 ──► 加载存档
    ├─► 新游戏 ────► 清空重开
    ├─► 设置 ──────► 设置面板 ──► 返回主菜单
    └─► 关于 ──────► 关于页面 ──► 返回主菜单
```
