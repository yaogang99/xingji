### 问题 #1: UI状态显示不完整

**描述**: `ExpeditionPanel.tsx` 缺少对 `locked` 状态的显示  
**影响**: 低 - 用户界面显示  
**修复**: 添加 `locked` 状态的中文显示 "🔒 锁定"

**修复代码**:
```typescript
function getStatusText(status: string): string {
  const texts: Record<string, string> = {
    idle: '待命',
    expedition: '探险中',
    returning: '返航中',
    locked: '🔒 锁定',  // 新增
  }
  return texts[status] || status
}
```
