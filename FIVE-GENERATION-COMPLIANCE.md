## 什么是五代原则？

五代原则（Five-Generation Principle）是航天级游戏设计文档的核心规范：

```
┌─────────────────────────────────────────────────────────────┐
│                    五代原则金字塔                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    第一代：数据层                             │
│              Data Layer - 所有实体完整定义                    │
│                                                             │
│              第二代：逻辑层                                    │
│         Logic Layer - 算法、公式、状态机                      │
│                                                             │
│              第三代：表现层                                    │
│      Presentation Layer - UI/UX、视觉、音效                   │
│                                                             │
│              第四代：验证层                                    │
│        Validation Layer - 测试、边界、异常                    │
│                                                             │
│              第五代：迭代层                                    │
│       Evolution Layer - 扩展、Mod、更新路径                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
